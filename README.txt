# iPhone Sales Analysis Dashboard

## Project Overview
Interactive dashboard analyzing 10,000+ iPhone sales records using Power BI, Python, and MySQL.

## Features
- Revenue & Profit by Region and iPhone Model
- Monthly Revenue and Profit Trends
- Discount vs Profit Scatter Analysis
- Interactive slicers for Region and iPhone Model

## Insights
- iPhone 15 Pro Max dominates sales
- Europe leads in revenue and profit
- High discounts significantly reduce profit

## Tools
- Python (Pandas) for data cleaning
- MySQL for data storage
- Power BI for interactive visualization

## How to Use
1. Open `iPhone_Sales_Dashboard.pbix` in Power BI Desktop
2. Explore charts and slicers
